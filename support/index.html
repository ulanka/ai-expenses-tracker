About the app

AI Expenses Tracker is a personal finance application that allows users to:

  • Record expenses and income
  • Upload receipt photos
  • View financial reports
  • Use AI-powered extraction and summaries
  • Access subscription-based features

The app is provided for personal financial tracking only.

Before reporting a problem

  • Make sure you are using the latest version of the app from the App Store
  • Restart the app
  • Use Restore Purchases if your subscription is not active

How to report an issue

To help us resolve your issue as quickly as possible, please include:

  • Clear step-by-step instructions to reproduce the problem
  • Screenshots or a short screen recording
  • Your device model and iOS version

We are constantly improving

We regularly release updates and improvements. If something is not working as expected, please contact us before leaving a review so we can help resolve the issue.
